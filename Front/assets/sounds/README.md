# Sons d'appel

Placez ici les fichiers audio (MP3) pour les appels :

- **ring_back.mp3** : son joué côté appelant en attendant que l'autre décroche (boucle).
- **incoming_call.mp3** : son joué côté appelé pour l'appel entrant (boucle).

Sans ces fichiers, les appels fonctionnent mais sans son de ringtone.

Vous pouvez utiliser des sons libres de droit (ex. Freesound.org, Pixabay) ou enregistrer des courts extraits.
