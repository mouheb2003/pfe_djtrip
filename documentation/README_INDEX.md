## 🎯 Guide d'utilisation de cet index

### Pour un nouveau développeur

1. Commencez par lire **`documentation/README.md`**
2. Suivez le guide **`documentation/SETUP.md`**
3. Consultez **`documentation/ARCHITECTURE.md`** pour comprendre la structure
4. Utilisez **`documentation/API_REFERENCE.md`** pour l'intégration API

### Pour comprendre le Backend

1. Lisez **`Back/README.md`**
2. Consultez **`documentation/ARCHITECTURE.md`** section Backend
3. Référez-vous à **`Back/EMAIL_VERIFICATION_SETUP.md`** pour l'email

### Pour comprendre le Frontend

1. Lisez **`Front/README.md`**
2. Consultez **`Front/FLUTTER_GUIDE.md`**
3. Référez-vous à **`Front/TOKEN_AUTO_REFRESH_GUIDE.md`**

### Pour résoudre des problèmes

1. Consultez les **CHANGELOG** récents
2. Vérifiez **`GOOGLE_SIGNIN_FIX_2026-03-08.md`** pour Google Sign-In
3. Consultez **`GOOGLE_FACEBOOK_AUTH_SETUP.md`** pour OAuth
